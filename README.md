# 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/SkyHypr.git

cd SkyHypr
```

Make installer executable

```bash
chmod +x install.sh
```

Run installer

```bash
./install.sh
```

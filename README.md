# SkyHypr

Minimal black/cyan Hyprland setup with custom utilities.

![Desktop](screenshots/desktop.png)

## Features

- Hyprland
- Waybar
- Kitty
- Rofi
- Fastfetch
- Cava
- Dolphin styling
- Firefox CSS
- Custom Wallpaper Picker
- Custom Power Menu
